# Dependencies
1. Python
2. Library:
   - mediapipe
   - open-cv
   - pydirectinput

# How to use
1. Install Python
2. Install `pip` (if you haven't)
3. Install libraries:
   - `pip install mediapipe`
   - `pip install opencv-python`
   - `pip install pydirectinput`
4. Open Minecraft, and run the program `python pose.py`
